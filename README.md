## 📖 Descripción

Mi portafolio personal desarrollado con **Next.js 14 + TypeScript**, desplegado en Vercel. Muestra mis proyectos, habilidades y experiencia como desarrollador Full Stack. Diseño moderno con animaciones suaves.

## 🌐 Demo en vivo

👉 **[portafoliopz.vercel.app](https://portafoliopz.vercel.app)**

## ✨ Secciones

- 🏠 **Hero** — Presentación con animación
- 👨‍💻 **Sobre mí** — Descripción profesional y trayectoria
- 🛠️ **Skills** — Tecnologías y herramientas que domino
- 🚀 **Proyectos** — Mis proyectos más destacados con links
- 📬 **Contacto** — Información de contacto directo

## 🏗️ Stack Tecnológico

| Capa | Tecnología |
|---|---|
| Framework | Next.js 14 (App Router) |
| Lenguaje | TypeScript |
| Estilos | CSS Modules / CSS3 |
| Despliegue | Vercel |

## 🚀 Instalación

```bash
git clone https://github.com/jzaruma1209/portafoliopz.git
cd portafoliopz
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
